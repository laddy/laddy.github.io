mini.css

  • 配布先: http://minicss.org/
  • Github: https://github.com/Chalarangelo/mini.css
  • CSSフレームワーク、サイズの小ささやレスポンシブ、 他のライブラリに非依存でカスタマイズがやりやすい特徴を持つ
  • Bootstrapよりもシンプルでクラスの命名規則などが似ている

Setup

$ npm install mini.css
$ bower install mini.css

CDN

<link rel="stylesheet" href="https://gitcdn.link/repo/Chalarangelo/mini.css/master/dist/mini-default.min.css">
<link rel="stylesheet" href="https://cdn.rawgit.com/Chalarangelo/mini.css/v2.2.0/dist/mini-default.min.css">
